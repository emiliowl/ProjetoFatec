Sobre
Este software é um projeto de código aberto, desenvolvido pelos alunos da Faculdade de Tecnologia de Jundiaí com o objetivo de cumprir as avaliações propostas peça disciplina de Linguagem de Programação de Bancos de Dados, ministrada pelo professor Emerson.

O escopo deste software está em um sistema biométrico para reconhecimento de impressões digitais de professores e alunos, e através disto substituir de maneira satisfatória o antigo método de averiguação de presença.
